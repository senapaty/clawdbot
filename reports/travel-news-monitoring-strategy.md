### 5.3 Cost Control Measures

1. **Rate Limiting**
   - Brave Search free tier: 1 request/second
   - Space out searches by 2-3 seconds
   - 2000 queries/month limit

2. **Smart Fetching**
   - Only fetch full articles if headline is relevant
   - Skip paywalled content
   - Use snippets for initial filtering

3. **Deduplication**
   - Store all findings in `memory/YYYY-MM-DD-travel-news.md`
   - Check against past alerts before notifying
   - Don't report same news twice

4. **Batch Processing**
   - Run all searches in parallel where possible
   - Summarize once at end
   - Single commit to GitHub

5. **Selective Alerting**
   - Only alert on actionable changes
   - "Nice to know" vs "Must act now"
   - Respect quiet hours (11 PM - 8 AM IST)

### 5.4 Token Efficiency

**Estimated costs per monitoring run:**
- 9 web searches: ~900 tokens
